安装哪吒面板的机器是RN的一台小鸡，本来安装的是ubuntu系统，想换成debian系统，所以需要对哪吒面板的数据进行迁移，所以写此篇文章记录迁移过程。

迁移准备

参考哪吒面板官方的wiki，所有的数据都存放在/opt/nezha目录下，所以需要备份一下此目录，下载到本地。

如何进行数据迁移、备份恢复？#

  1. 先运行一键脚本，选择停止面板
  2. 在旧服务器中打包/opt/nezha文件夹，复制到新环境相同位置解压
  3. 在新环境中运行一键脚本，选择启动面板

迁移

将备份数据上传到新的服务器，解压到/opt/nezha目录下。通过nezha.sh启动面板。

    }
}
